# Third-Party Data Notices

This project downloads the NLTK `names` and `cmudict` data packages at the
pinned commit identified in `run_evaluation.py`. The project does not include
the source ZIP files, but its local outputs contain derived names and phoneme
references. Preserve this notice and the two upstream README copies with any
copy of those outputs:

- [`UPSTREAM_NAMES_README.txt`](UPSTREAM_NAMES_README.txt)
- [`UPSTREAM_CMUDICT_README.txt`](UPSTREAM_CMUDICT_README.txt)

## Names Corpus, Version 1.3 (1994-03-29)

Copyright (C) 1991 Mark Kantrowitz  
Additions by Bill Ross

This corpus contains 5001 female names and 2943 male names, sorted
alphabetically, one per line.

You may use the lists of names for any purpose, so long as credit is given in
any published work. You may also redistribute the list if you provide the
recipients with a copy of this README file. The lists are not in the public
domain (the author retains copyright on the lists) but are freely
redistributable. The source README requests that additions be sent to Mark
Kantrowitz at `mkant+@cs.cmu.edu` and identifies the original repository as:
<http://www-2.cs.cmu.edu/afs/cs/project/ai-repository/ai/areas/nlp/corpora/names/>.
